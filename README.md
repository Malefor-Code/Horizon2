# Narrative Companion

A small offline-capable narrative companion for tabletop games built with React + TypeScript + Vite.

Quick start:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Open http://localhost:5173

Notes:

- The app registers a basic service worker for offline support.
- Drop JSON story files via the file picker to load custom stories.
